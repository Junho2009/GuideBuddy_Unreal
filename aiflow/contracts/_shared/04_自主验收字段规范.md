# 自主验收字段规范

Verifier 中每条检查至少包含：

- `id`：稳定检查 ID。
- `title`：人类可读标题。
- `automation_level`：`auto` / `manual` / `hybrid`。
- `blocking`：是否阻塞放行。
- `stimulus`：如何触发。
- `observe`：观察什么。
- `oracle`：什么算通过。
- `evidence`：证据写到哪里。
- `on_fail`：失败后如何处理。

## automation_level

- `auto`：AI 必须真实执行并读取结果。
- `manual`：需要人类判断的烟测项。
- `hybrid`：AI 收集证据，人类确认主观质量。

## MVP01 自动验收重点

- 事件文件存在。
- JSON / JSONL 可解析。
- 至少包含尝试开始、玩家输入、敌人动作、伤害或死亡。
- `attempt_summary.json` 能引用死亡前关键事件。
- 输出路径不会覆盖上一轮 run。
