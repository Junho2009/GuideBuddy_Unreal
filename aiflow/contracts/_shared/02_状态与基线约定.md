# 状态与基线约定

## 合同状态

- `draft`：草案，可讨论，可修改，不允许作为正式实现授权。
- `proposed`：已形成推荐默认项，等待人类确认。
- `confirmed`：人类已确认目标、范围和验收口径。
- `frozen`：可作为正式执行基线，除 Change Request 外不再随意改动。
- `active`：当前正在执行。
- `accepted`：已通过 verifier 并被 ledger 接受。
- `stale`：已被新 baseline 替代。
- `rejected`：确认不再采用。

## 切换 Current Execution Baseline 的条件

必须同时满足：

1. Brief 已 `confirmed` 或 `frozen`。
2. Task Pack、Manifest、Verifier 引用同一个 baseline。
3. 人类明确同意按该 baseline 推进。
4. `aiflow/CONTEXT.md`、`aiflow/specs/GAME_PHASES.md` 与对应 MVP phase 文档同步。
