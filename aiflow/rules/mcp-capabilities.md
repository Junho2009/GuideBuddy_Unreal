# UE / 资产 / MCP 能力规则

适用场景：

- 需要检查 Unreal 资产引用。
- 需要操作 Blueprint、地图、输入或编辑器资产。
- 需要通过 MCP 或等价工具读取编辑器状态。

## 原则

- 能用代码或文本完成的逻辑，不新增到二进制 Blueprint 图。
- 资产和 Blueprint 可以作为既有战斗内容与配置来源。
- 修改资产前先确认当前合同允许该范围。
- 自动化工具已覆盖的检查，不改写成人工步骤。

## MVP01 默认边界

- 可以只读调查 `SampleDemoShowcaseMap` 和相关战斗资产。
- 可以接入或配置遥测所需的最小启用路径。
- 不重写现有战斗行为。
- 不改变战斗数值和平衡。
